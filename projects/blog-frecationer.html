Blog Frecationer

Platform blog komunitas yang dirancang untuk menampilkan artikel, informasi, dan konten digital dengan tampilan modern dan responsif.

Tentang Proyek

Blog Frecationer merupakan website blog komunitas yang digunakan untuk mempublikasikan artikel, berita, serta konten digital lainnya. Proyek ini berfokus pada penyajian konten yang rapi, navigasi yang mudah, serta tampilan yang responsif di berbagai perangkat.

Peran Saya di Proyek Ini

Pada proyek ini, saya bertanggung jawab dalam pengembangan bagian frontend website. Saya membangun struktur layout, komponen tampilan, serta memastikan desain dapat berjalan dengan baik di berbagai ukuran layar menggunakan Tailwind CSS.

Kontribusi Frontend

✔ Pembuatan layout halaman blog yang responsif
✔ Desain kartu artikel dan struktur konten
✔ Navigasi yang adaptif untuk mobile dan desktop
✔ Styling modern menggunakan Tailwind CSS

Teknologi yang Digunakan

HTML Tailwind CSS Responsive Design
