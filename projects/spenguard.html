SpendGuard

Aplikasi web pengelola keuangan pribadi yang saya bangun berdasarkan pengalaman pribadi dalam mengatur pemasukan, pengeluaran, dan kondisi finansial sehari-hari.

Latar Belakang Ide

SpendGuard berawal dari kebutuhan pribadi saya dalam mengelola keuangan sehari-hari. Saya sering merasa kesulitan melihat alur pemasukan dan pengeluaran secara jelas, sehingga sulit memahami kondisi finansial secara keseluruhan. Dari masalah tersebut, saya merancang aplikasi ini sebagai solusi untuk mencatat transaksi, memantau saldo, serta menampilkan gambaran keuangan secara visual dan terstruktur.

Tentang Proyek

SpendGuard merupakan aplikasi manajemen keuangan berbasis web yang memungkinkan pengguna mencatat transaksi, memantau saldo dompet, serta menganalisis pola pengeluaran. Proyek ini menekankan pada struktur data yang rapi, visualisasi grafik yang informatif, dan antarmuka yang responsif di berbagai perangkat.

Fitur Utama

✔ Pencatatan pemasukan dan pengeluaran
✔ Monitoring saldo dompet secara real-time
✔ Dashboard dengan grafik keuangan interaktif
✔ Tampilan responsif berbasis Tailwind CSS

Teknologi yang Digunakan

Django Tailwind CSS PostgreSQL Chart.js
